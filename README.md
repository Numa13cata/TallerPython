# TallerPython
En este repositorio se incluyen 9 archivos que contienen el desarrollo completo de un taller de Python. Cada cuaderno aborda un tema específico:

El primer archivo introduce el manejo de cadenas de texto en Python.

El segundo explica el uso de las tuplas.

El tercero presenta el trabajo con listas.

El cuarto desarrolla el uso de conjuntos, destacando sus diferencias frente a las listas.

El quinto se centra en los diccionarios, describiendo su estructura y principales funciones.

El sexto aborda los condicionales y operadores de comparación como <, >, ==, !=, entre otros.

El séptimo explica los bucles en Python, especialmente while y for.

El octavo describe las funciones, incluyendo su definición, llamada y el manejo de parámetros.

El noveno introduce las clases en Python, con ejemplos basados en figuras geométricas, se relaciono mucho con la programación orientada a objetos.

Bono: Se añade una colección de ejercicios que integran y resumen los contenidos trabajados en los nueve archivos anteriores.
